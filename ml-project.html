Stock Price Prediction using Machine Learning

DTSC691 Capstone Project | December 2025

Project Overview

This project implements a comprehensive machine learning pipeline to predict short-term stock price movements based on historical market data. Using 10 years of Apple Inc. (AAPL) stock data, the project demonstrates the application of technical analysis indicators and machine learning algorithms to forecast daily price direction.

Problem Statement

Challenge: Can machine learning identify patterns in historical stock price data to predict short-term price movements?

Financial markets are influenced by countless factors, making price forecasting difficult but valuable for data-driven decision-making. This project explores whether patterns within historical market data‚Äîsuch as price momentum, volatility, and trading volume‚Äîcan be leveraged to predict short-term stock movements.

Approach & Methodology

1. Data Collection

  • Source: Yahoo Finance API
  • Ticker: AAPL (Apple Inc.)
  • Period: January 2015 - December 2025 (10 years)
  • Samples: 2,704 trading days

2. Feature Engineering (28 Technical Indicators)

  • Trend Indicators: SMA, EMA, Price-to-SMA ratios
  • Momentum Indicators: MACD, RSI, Stochastic Oscillator
  • Volatility Measures: Bollinger Bands, ATR, Rolling Volatility
  • Temporal Features: Lagged returns, momentum calculations

3. Model Development

  • Trained multiple models: Logistic Regression, Random Forest, XGBoost
  • Applied time-series cross-validation to prevent data leakage
  • Implemented regularization techniques to mitigate overfitting
  • Selected Logistic Regression as best model (F1-Score: 0.59)

4. Model Evaluation

  • Statistical Metrics: Accuracy (51%), Precision (0.54), Recall (0.68)
  • Financial Backtesting: Simulated trading with transaction costs
  • Interpretability: SHAP analysis to identify key predictive features

Results & Key Findings

Model Performance

The Logistic Regression model achieved 51% accuracy in predicting next-day price direction, marginally better than random guessing (50%). While this demonstrates the inherent difficulty of short-term stock prediction, the model successfully identified weak but consistent patterns in technical indicators.

Financial Performance (Backtesting)

When simulated in a trading strategy, the ML model underperformed a simple buy-and-hold approach:

  • ML Strategy Return: -14.34%
  • Buy & Hold Return: +58.64%
  • Key Insight: Transaction costs and missed opportunities significantly impact active trading strategies

Feature Importance

SHAP analysis revealed that recent price momentum (Daily_Return), volatility measures (ATR), and short-term moving averages (SMA_10) were the most influential features in model predictions.

Critical Insights

  • Technical indicators alone provide limited predictive edge in efficient markets
  • Transaction costs significantly erode returns in active trading strategies
  • Passive investing often outperforms algorithmic approaches for individual investors
  • Demonstrates both the capabilities and fundamental limitations of ML in finance
